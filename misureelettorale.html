📋 Regole Generali per i Materiali

📁

Formati accettati

JPG, PNG, GIF (per animazioni). Non sono accettati file PSD, AI, PDF o video.

🎨

Modalità colore

Tutti i file devono essere in modalità RGB. I file in CMYK (stampa) non sono compatibili con il web.

⚖️

Peso massimo

Rispettare i limiti di peso indicati per ogni formato. File troppo pesanti rallentano il caricamento della pagina.

🔍

Testi leggibili

Assicurarsi che tutti i testi siano leggibili anche su dispositivi mobile. Evitare font troppo piccoli.

⚖️

Normativa elettorale

Il materiale deve rispettare la normativa vigente sulla propaganda elettorale. La redazione verificherà la conformità.

📅

Tempi di consegna

I materiali devono essere inviati almeno 48 ore prima della data di pubblicazione richiesta.

📤

Come inviare i file

✅

Approvazione

La redazione si riserva il diritto di rifiutare materiali non conformi agli standard tecnici o editoriali.
